CORS چیست ؟ 

اگر تا حالا از AJAX استفاده کردین حتما با خطای زیر مواجه شدین :

توی این پست میخوایم ببینیم این ارور چرا رخ میده و اصلا CORS چی هست.

 

CORS چیست ؟

CORS (Cross-origin resource sharing) یک مکانیزم بر مبنای هدرها (Header) ست که برای مرورگر تعیین میکنه که بجز دامنه فعلی از چه دامنه های دیگه ای منابع به اشتراک بگذاره.یعنی این مکانیزم نمیذاره که هکر ها از یک سایت به یک وبسایت دیگه ریکوئست الکی بدن.

چرا ؟

تصور کنید روی یه وبسایتی چند تا جای تبلیغات داره. حالا یه هکر عزیز میاد و روی یکی از جای این تبلیغ ها، تبلیغ خودشو میذاره. ولی همراه با تبلیغش یک اسکریپت هم بارگذاری میکنه ..‌. این اسکریپت به محض باز شدن سایت یه درخواست به: 

https://mybank.xyz/delete

میفرسته.

اگر قبل از باز کردن این وبسایت توی mybank.xyz لاگین کرده باشین، از اونجایی که session ها هم ارسال میشن، اکانت شما (توی سایت بانکتون) پاک میشه !!

ولی مرورگر شما قبل از فرستادن درخواست با توجه به هدر های CORS میبینه که وبسایتی مثلا مثل زومیت اجازه درخواست فرستادن به مثلا بانک ملت رو نداره و اون درخواست رو نمی فرسته.

 

چطوری استفاده کنم ؟

 

برای این که فقط از مبدا های مشخص درخواست دریافت کنید، کافیه این هدر رو روی صفحات مورد نظر بفرستید : 

Access-Control-Allow-Origin: http://www.example.com
